# Yandex Cloud Managed ClickHouse Cluster

- Create a Managed ClickHouse cluster with a list of users with settings and a list of databases with permissions.
- Define config of cluster.

## ClickHouse cluster definition

ClickHouse module requires a following input variables:

- VPC network id.
- VPC network subnet id.
- Hosts - a list of hosts with host type, zone name and subnet id.
- Databases - a list of databases with database name.
- Users - a list of users with a list of grants to databases.
